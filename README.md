# Bootstrap 4 and Sass Starter Pack

### Whats included?

- Gulp tasks
- Sass Compiler
- BrowserSync
- Bootstrap
- jQuery and Popper.js

### How to start dev server?

1. Download Project.
2. In root folder run `npm install`
3. To run gulp task run `npm start`
4. Open browser to `localhost:3000`
5. Start Developing!
